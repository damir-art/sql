# Параметры столбцов
https://metanit.com/sql/mysql/2.4.php  

- PRIMARY KEY - первичный ключ
  - по этому ключу идентефицируются строки таблицы, все они должны быть уникальны
- AUTO_INCREMENT - автоматические увеличение на единицу
- UNIQUE - уникальность занчения (для PK устанавливать не нужно)
- NOT NULL - ячейка не должна быть пустой
- атрибуты
  - UNSIGNED - число всегда должно быть положительным (для PK можно не устанавливать, но для оптимизации можно включить, чтобы в этой ячейке не хранились ресурсы для отрицательных чисел)
